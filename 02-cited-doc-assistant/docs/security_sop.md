# Information Security Standard Operating Procedure

Document ID: SEC-SOP-014
Owner: Security Operations

## Password Policy

Passwords must be at least 14 characters and include uppercase, lowercase, numbers, and symbols. Passwords expire every 90 days. Reuse of the last 12 passwords is prohibited.

## Incident Reporting

Security incidents must be reported to security@company.com within one hour of discovery. Critical incidents require immediate phone escalation to the on-call security engineer.

## Data Classification

Public data may be shared externally without approval. Internal data requires manager approval before external sharing. Confidential data must be encrypted at rest and in transit.

## Access Reviews

System owners must review user access quarterly. Dormant accounts inactive for 45 days are automatically disabled.

## Phishing Response

If an employee clicks a suspicious link, they must disconnect from the network, report the event to the help desk, and change their password immediately.
